Tools for Generic Purpose
Tableau
  • Intuitive to use, even for business users
  • Stroy telling feature
  • Connectivity with various data sources
QlikView
  • Associative technology and filters
  • Ability to compress data and act as warehouse
  • Fast deployment cycle
QlikSense
  • Drag and drop interface with power of Qlik associative technology
  • Story telling feature
  • Easy to use even for business users
D3.js
  • Provides interactive visualizations over the web
  • Open Source
  • Exhaustive libraries for visualization
R Programming
  • Larger data science and modeling capabilities
  • Open source, highly engaged community
  • Ability to create interactive visualizations
MicroStrategy
  • Desktop version free for download
  • Highly scalable - can handle Petabytes of data
  • Enables self service analytics
Weka
  • Eceptional ability to generate simple plots
  • Primary used for data mining tasks
  • Provides a powerful way to explore data
SAS
  • Primarily used for predictive modeling
  • Scalable, but could be costly
  • Made significant improvements with SGPLOT
SAS VA
  • Drag and drop interface
  • Good for data exploration before predictive modeling
  • Includes correlation plots and forecasting

Tools for Specific Purpose
Instant Atlas
  • Design highly interactive reports
  • Design to convey information quickly
  • Handy templates for maps and charts
WolframAlpha
  • A computational knowledge engine
  • Intelligently display charts
  • Requires no configuration
Cytoscape
  • Advanced analysis and modeling using apps
  • Visualize human-curated pathway datasets
  • Visualize social netowrks or interpersonal relationships
NetworkX
  • Generate classic graphs, random graphs, syntehtic networks
  • Nodes contains any media type (images and XML)
  • Supports cross-compatible data visualizations
Gephi
  • Built-in 3D rendering engine
  • Deep data analysis to examine relationships
  • Widely used to visualize social networks
